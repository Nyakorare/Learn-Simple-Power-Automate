Hands-on path

Try this in Power Automate
  • In Power Apps Studio, add a button; wire it to create or run an instant cloud flow.
  • In the flow, use the Power Apps trigger; define parameters the app will pass.
  • Optional: add Respond to Power Apps to return values to the app.
  • Publish and play the app; tap the button and confirm the flow run.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
