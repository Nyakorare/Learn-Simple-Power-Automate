Hands-on path

Try this in Power Automate
  • Create a Microsoft Form with one or two questions and note its name.
  • In Power Automate, use When a new response is submitted and select that form.
  • Add Get response details; map answers into email, SharePoint, or Excel.
  • Submit a test response and confirm a new run appears with the expected outputs.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
