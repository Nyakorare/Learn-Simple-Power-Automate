Hands-on path

Try this in Power Automate
  • On a list you own, create → Automated → When an item is created or modified; select site and list.
  • Add Get item using the ID from the trigger dynamic content.
  • Add Update item or Send an email using columns from the item.
  • Create or edit a list item in SharePoint and confirm the flow run in history.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
