Hands-on path

Try this in Power Automate
  • Add action → Initialize variable. Name it Counter, type Integer, value 0.
  • Add Increment variable (or Set variable) targeting Counter inside the same flow.
  • Add Compose; open Expression and reference variables with variables('Counter') or pick from Dynamic content if listed.
  • Test the flow and verify the variable value inside the run details for each step.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
