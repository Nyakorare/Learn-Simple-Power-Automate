Hands-on path

Try this in Power Automate
  • Ensure you have an array (from a connector output or Parse JSON). Add Apply to each and select that array.
  • Drop one action inside the loop, such as sending mail or updating a row per item.
  • Open Apply to each settings and keep Concurrency at 1 until you understand downstream limits.
  • Test with two or three items first; expand the list after runs stay fast and stable.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
