Hands-on path

Try this in Power Automate
  • Add an action and pick a connector you have not used; complete sign-in when prompted.
  • Review Data → Connections (or equivalent) to see the new connection object.
  • Compare actions labeled Standard versus Premium in the picker and note licensing implications.
  • Inside a solution, add a connection reference before exporting so imports map cleanly in another environment.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
