Hands-on path

Try this in Power Automate
  • Open a flow and select the first card. Read required fields and optional advanced options.
  • Add a second action. In a text field, open Dynamic content and insert a value produced by the trigger.
  • Rename the trigger and action from the … menu so run history shows clear labels.
  • Run Test → Manually, complete the run, and click each step to compare INPUTS and OUTPUTS.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
