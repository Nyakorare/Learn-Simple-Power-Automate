Hands-on path

Try this in Power Automate
  • Ask your admin (or check Power Platform admin center) which Power Automate plans and premium connectors your tenant allows.
  • Build a tiny flow using only standard connectors, run it once, and open run history to see a successful run counted.
  • Add one action that shows a premium badge (for example Dataverse). Read the licensing hint in the designer if it appears.
  • Note per-user versus per-flow concepts at a high level and record where to request licenses in your org.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
