Hands-on path

Try this in Power Automate
  • Open make.powerautomate.com and sign in with your work or school account.
  • Use the environment picker in the header to select a sandbox or trial environment (avoid production while experimenting).
  • Select Create, skim Instant cloud flow and Automated cloud flow, and open a template preview without saving if you only want to explore.
  • From My flows, open any flow to see the designer: one trigger at the top, actions below, and the mini-map on the side.
  • Close the tab or save a draft when you are done—no change is required on this site.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
