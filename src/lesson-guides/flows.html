Hands-on path

Try this in Power Automate
  • Create → Automated cloud flow → choose Manually trigger a flow (or any simple trigger) → Create.
  • Add one action such as Compose or Send an email, name the flow, and Save.
  • Create → Instant cloud flow → pick Manually trigger or Power Apps to see a user-started pattern.
  • Create → Scheduled cloud flow, set a daily or hourly recurrence, and Save to inspect the recurrence trigger card.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
