Hands-on path

Try this in Power Automate
  • Insert Condition after your trigger. Build a simple comparison (for example a field equals a text literal).
  • Place one action in If yes and a different action in If no.
  • Replace the condition with Switch when you have several discrete values to branch on.
  • Save; run twice with inputs that hit yes and no (or different switch cases) and confirm both paths.

Use a dev or trial environment. Designer labels change over time—use the search box in the add-action panel if a name differs slightly.
