Types of flows

  • Automated cloud flow — starts when an event happens in a connector.
  • Instant cloud flow — user or app starts it manually (button, Power Apps, Teams).
  • Scheduled cloud flow — runs on a recurrence.
  • Desktop flow (RPA) — automates Windows/UI through Power Automate for desktop.
  • Business process flow — guided stages in model-driven apps (Dataverse).
