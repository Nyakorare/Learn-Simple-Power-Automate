Microsoft Forms

Forms is a great entry point for beginner automations: collect responses, then immediately route or store the data.

Baseline Forms automation
  • Trigger: When a new response is submitted.
  • Get data: Get response details to expand all answers.
  • Use data: classify responses (urgent/non-urgent), normalize text, and check completeness.
  • Output: create a SharePoint item, send a Teams message, or notify approvers.
Helpful data handling lessons
  • Use Condition for branching logic based on answers.
  • Use Compose for clean output text and labels.
  • Use Parse JSON when a form answer includes structured data from another system.
  • Use a fallback branch for missing or invalid responses.

Tip: rename steps by intent (Get form answers, Build manager message) so future debugging is easier.
